## 使用方式

1. 安装插件后, 插件入口会出现在 `VSCode` 侧边栏
2. 打开插件侧边栏上面有 `书源编辑` 按钮, 点击后可以配置源
3. 配置好源后, 侧栏列表会出现配置好的源。
